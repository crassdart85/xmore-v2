Xmore Documentation

Xmore is an AI-powered stock prediction system for the Egyptian Exchange (EGX) and global markets. This page documents every feature available on the dashboard and Xmore Pro.

Overview

Live

Xmore aggregates signals from 5 AI agents — Moving Average, RSI, Volume Spike, Random Forest ML, and Gemini LLM — into a unified consensus score. The dashboard auto-refreshes data on every EGX trading day.

🤖 5 AI Agents

MA Crossover, RSI, Volume Spike, Random Forest, and Gemini LLM each vote independently with weighted confidence.

📊 Xmore Score

0–100 composite bullishness score. ≥65 = strong buy zone; ≤35 = bearish. Updated daily after market close.

🌐 Bilingual

Full English / Arabic toggle. All labels, signals, and dates switch language. Arabic uses RTL layout.

🌙 Dark / Light theme

Theme preference saved to localStorage and persists across sessions.

📱 Responsive

Mobile-optimised layout. Tables scroll horizontally on small screens. All modals are touch-friendly.

🔐 Auth

Sign Up / Sign In with email. JWT stored in HTTP-only cookie. Portfolio, alerts, and forecasts are per-user.

Quick start
Visit xmore-project.onrender.com → sign up for free → add stocks to your watchlist → check the Predictions tab each morning before market open (09:00 Cairo).

📡 Consensus Signal

BUY / HOLD / SELL derived from majority vote across all active agents. Displayed with conviction label (Strong/Moderate/Weak).

🔢 Xmore Score

0–100 bull/bear composite. Color-coded: green ≥65, red ≤35, grey in-between.

🔍 Screener

Filter by signal (BUY/SELL), confidence threshold, sector, and volume. Results sortable by any column.

📰 Sentiment Badge

Colour-coded news sentiment (Positive / Neutral / Negative) beside each stock, derived from Gemini LLM analysis of latest headlines.

How to read a signal row

  1. Signal — BUY means majority of agents predict price will rise within 5 trading days.
  2. Confidence — weighted average certainty across agreeing agents (0–100%). Higher = more conviction.
  3. Xmore Score — bull pressure index. Score of 80 means 80% of composite weight is bullish.
  4. Agents column — e.g. "4/5" = 4 of 5 agents voted for the consensus direction.
  5. Price — last closing price in EGP from EGX or TradingView Scanner.

Per-stock AI Brief

New

Every stock can generate a 3-sentence Gemini-powered analyst brief covering current stance, key risk, and short-term outlook. Briefs are cached for 1 hour to minimise API cost.

AI Brief Modal — COMI
COMI
Generated by Gemini 2.5 Flash · Cached 1 hour

1. Stance: COMI carries a strong BUY signal with 87% confidence, underpinned by bullish MA crossover alignment and elevated RSI momentum — 4 of 5 agents agree on the upside case.

2. Key risk: The primary downside risk is broad EGX market sentiment reversal; COMI has historically tracked EGX30 beta closely, so any index weakness could compress the position within the 5-day horizon.

3. Outlook: Short-term momentum favours continuation toward the 64–66 EGP resistance zone, provided the current volume profile holds above the 20-day average.

Availability
AI Brief requires GOOGLE_API_KEY to be configured. Briefs are accessible from the Comparison Tool modal and from individual stock rows in Predictions (via the 📄 icon when visible).

Forecast Portfolios

Forecasts Tab

Build a named portfolio with a custom horizon (1m to 2y) and Xmore will forecast each stock's expected return using all active agents. Daily actuals are recorded and compared against the forecast.

📋 Custom Portfolios

Create multiple named portfolios (e.g. "Conservative Q2", "Aggressive 6M") each with its own stocks and horizon.

📅 Six Horizons

1 month (21 trading days), 2m, 3m, 6m, 1 year, 2 years. Forecast auto-runs on creation.

📈 Daily Actuals Tracking

A cron records the actual closing price each day. The chart overlays forecast vs actual return over time.

📝 Business Narrative

Auto-generated plain-English summary: scenario, phase (early/mid/late), avg forecast gap, and per-position beat count.

Macro Brief

Pro + Chat

A Gemini 2.5 Flash grounded web search producing a structured Egyptian macro overview: EGX index level, USD/EGP rate, inflation, interest rate, key market movers, and 3 web source citations.

🔍 Grounded web search

Uses Google Search grounding to pull real-time macro data — not just the training cutoff.

🔗 Web source pills

Clickable blue citation pills for each web source used in the response.

🔄 Auto-refresh

Macro Brief auto-loads on Pro page open and refreshes every 60 minutes via setInterval.

💬 Chat quick chip

"📊 Macro Brief" quick chip in the Market Assistant chat widget also triggers the macro endpoint.

Market Assistant

Chat Widget

An AI chat widget powered by Gemini with full context injection: live prices, consensus signals, sentiment, ETF prices, and — when logged in — your personal forecast portfolio data.

📊 Live market context

Every message includes today's gainers/losers, volume leaders, sentiment, and consensus signals as background context.

🔮 Portfolio context

When signed in, the assistant knows your forecast portfolios and can answer questions about positions vs forecasts.

📰 News-aware

Arabic-aware search extracts keywords from your question to find relevant recent headlines before answering.

⚡ Quick chips

"📊 Macro Brief", "📈 Top Movers", "🎯 Buy Signals" — one-tap prompts for common questions.

🛠

Admin Panel

Admin Only

Accessed at /admin. Requires ADMIN_USERNAME + ADMIN_PASSWORD credentials (JWT-protected). Contains system health, report upload, RAG embedding status, ETF documents, and user reports.

📤 Report Upload

Upload PDF/image market reports (up to 25 MB). Text extracted via pdf-parse; images OCR'd via tesseract.js. Auto-embedded if GOOGLE_API_KEY is set.

🧠 Ask Reports (RAG)

Semantic search over embedded market report chunks. Shows AI answer + source document citations.

💊 System Health

Live DB row counts, latest prediction date, agent status, and pending embedding jobs.

📄 ETF Documents

View all uploaded ETF factsheets in the rag_document table with embedding job status.

Time Machine

History Tab

Replay any past trading session. Select a historical date to see what the AI agents predicted that day, alongside actual price outcomes — useful for backtesting your own strategy.

📅 Date picker

Select any past EGX trading day. Predictions and prices are loaded from the database snapshot for that date.

✅ Outcome overlay

Green ✓ / Red ✗ icons show which signals were correct — the actual price move is shown alongside.

Overall accuracy, best and worst calls for the selected date, per-agent win rate for that session.

🏋️ Backtest mode

API Reference

REST

All endpoints return JSON. Auth endpoints require a valid session cookie. Admin endpoints require Authorization: Bearer <token>.

# ── Predictions & Consensus ── GET /api/consensus → array of all stocks with signals GET /api/stocks/:symbol/brief → {brief} Gemini 3-sentence AI narrative GET /api/signal-accuracy?horizon=10 → per-stock accuracy at D+5/D+10/D+20 # ── Prices ── GET /api/prices → latest close price for all symbols GET /api/prices/:symbol → price history for one symbol GET /api/sentiment → latest sentiment per stock GET /api/stats → {total_stocks, latestDate, …} # ── Portfolio & Trades ── GET /api/trades/portfolio → {open_positions, closed_positions, stats, totals, sector_breakdown} POST /api/trades/positions → add position {symbol, entry_price, quantity, entry_date} GET /api/trades/alerts → list alerts (auto-triggers active ones) POST /api/trades/alerts → create alert {symbol, condition, target_price} DELETE /api/trades/alerts/:id → delete alert # ── FX & Rates ── GET /api/fx-rates → {USD_EGP, USD_SAR, SAR_EGP, XAU_USD, …} GET /api/fx-rates/history?days=90 → array of daily rate rows (oldest first) # ── Forecast Portfolios ── GET /api/portfolio-forecasts → list user portfolios POST /api/portfolio-forecasts → create portfolio + run forecast GET /api/portfolio-forecasts/:id/results → per-stock forecast results GET /api/portfolio-forecasts/:id/chart → forecast vs actuals time series # ── ETF ── GET /api/etf/instruments → all ETF instruments GET /api/etf/prices/:symbol → price history GET /api/etf/nav/:symbol → NAV history GET /api/etf/holdings/:symbol → holdings snapshot GET /api/etf/country-exposure → country weights # ── RAG / Chat ── POST /api/rag/chat → {message} → {reply} market assistant POST /api/rag/macro → {} → {answer, sources[]} grounded macro brief POST /api/rag/ask → {question} → {answer} report RAG (admin) # ── Auth ── POST /api/auth/signup → {email, password} → sets cookie POST /api/auth/login → {email, password} → sets cookie POST /api/auth/logout → clears cookie GET /api/auth/me → {id, email} if logged in
Auth note
Portfolio, alerts, and forecast endpoints require a valid session cookie (set by /api/auth/login). Pass credentials: 'include' on all fetch calls from the frontend.
