How It Works

Four layers. One recommendation.

Every signal goes through a full pipeline before it reaches you.

01
Data Collection
EGX prices, Arabic + English news, macroeconomic indicators (Brent crude, USD/EGP at CBE rates, emerging-market ETFs), and social sentiment — refreshed daily.
→
02
Signal Agents
Four independent agents vote: ML (LightGBM with Optuna tuning), RSI momentum, Moving Average crossover, and Sentiment. Each agent is weighted by its recent accuracy.
→
03
Risk Gating
Signals below 60% confidence are suppressed to HOLD. High-volatility regimes trigger tighter thresholds. GARCH-modelled volatility overrides simple rolling vol.
→
04
Regime Filter
HMM market-regime detection (Calm / Turbulent / Crisis) gates the final output. Crisis regime blocks all UP signals market-wide. Turbulent regime downgrades conviction.
